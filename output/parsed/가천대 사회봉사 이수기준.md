# 가천대 사회봉사 이수기준

> 출처: 가천ESG센터 사회봉사 교과목 운영 안내 (https://www.gachon.ac.kr/ESG/9032/subview.do)
> 카테고리: social_service/requirement
> 비고: 학기별 공지에서 시효성 정보(제출기간 등)는 제외하고 기본 이수기준만 정리함. 매 학기 제출기간·양식은 ESG센터 공지 확인.

1. 이수방법

1) 사이버캠퍼스 동영상 (장애이해교육, 기초소양교육) 시청

2) 봉사활동 30시간 ‘VMS'와 '1365 자원봉사센터'에서 발급된 봉사 확인증만 인정됩니다.

( 해당 기관에서 인정한 비대면, 온라인 봉사도 가능) -> 입학 시점 이후 30시간 누적

3) 소감문 제출 (서술식 줄바꿈 없이 25줄 이상, 조건 미달 시 제출 서류 반려)

2. 서류 제출방법

1) 봉사활동인증서(1365, VMS에서 다운로드), 사회봉사 교과목 서류 작성 -> 나이테 시스템 업로드.

※ 가천대학교 포털 -> 나이테시스템 -> ♡봉사활동 -> 사회봉사 교과목 서류 제출 에 작성

2) 봉사 인증서가 여러 장일 경우 30시간으로 취합하여 한 게시물로 업로드 할 것. (인증번호가 여러 개일 경우 대표적인 한 곳만 기입)

3) 연도/학기 입력 시 봉사 활동 기간이 아닌 교과목 수강 학기로 기입

- 교과구분 시 교양필수, 교양선택만 성적 반영.

- 개인봉사로 입력 시에 졸업인증 또는 학점 이수 불가.

※ 나이테 시스템에 서류 제출 시 해당 사회봉사 수강연도 / 학기 / 교과구분 ( 교필 , 교선 ) 확인 필수.

3. 학습자 유의 사항

1) 교양필수 사회봉사 이수자만 교양선택 사회봉사2 이수가 가능합니다.

- 사회봉사(선택) 과목 수강을 하였어도 사회봉사(필수)가 미이수일 경우 사회봉사 필수로 우선 처리

- 한 학기에 교양필수, 교양선택 중복 수강 불가

2) 헌혈 관련: 전혈은 1회 4시간, 최대 2회 인정, 성분 헌혈의 경우에는 1회 4시간, 최대 3회까지 인정

※ 헌혈로 인정 받을 수 있는 최대 봉사활동시간: 16시간

3) 청강생 신청 및 승인 불가, 수강 신청 기간에 반드시 수강 신청.

4) 졸업 유예자가 수강 신청이 필요할 경우에는 가천ESG센터로 연락 바랍니다.
