### 크롤링 방법 (`data_pipeline/crawl_gachon.py`)
- **대상**: `www.gachon.ac.kr` 정적 서버렌더링 페이지(`subview.do`), 게시글(`artclView.do`). 콘텐츠는 `#contentsEditHtml` / `div._obj`.
- **SSL**: 가천 서버가 레거시 cipher 요구 → 기본 `requests` 는 핸드셰이크 실패. `TLSAdapter` 로 `DEFAULT@SECLEVEL=1` 설정하여 해결.
- **정제**: CMS 템플릿 찌꺼기(`.jsp`/`WEB-INF`/템플릿 ID), 게시글 잡동사니(글번호·수정일·첨부파일·이전/다음글), 링크 경로 토큰(`/kor/3135/subview.do`) 필터링.
- **출력**: `output/parsed/<name>.md` (헤더에 출처 URL + 카테고리) + `_crawl_manifest.json`(source→category 매핑).
- **주의(환각 방지)**: `SOURCES` URL·전화번호는 전부 실제 확인된 값만 등록. 게시글 큐레이션본 2건(외국어 기준점수, 사회봉사 이수기준)은 재크롤이 덮어쓰지 않도록 `SOURCES` 에서 제외(유래 URL은 주석 보존).
- **학사일정**: 월별 AJAX/이미지 페이지라 HTML 크롤 대신 PDF 경로(`--pdf` → `parse_pdf.py`) 권장.

### 실행
```bash
python -m data_pipeline.crawl_gachon          # 전체 크롤
python -m data_pipeline.crawl_gachon --dry    # 저장 없이 미리보기
python -m data_pipeline.crawl_gachon --pdf    # 학사일정 PDF 다운로드
```
